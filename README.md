# FrontendBuddy

A futuristic AI-focused blog landing page showcasing the latest trends, spotlight deep dives, and curated newsletters for builders.

## Getting started

Open `index.html` in your browser to explore the experience. No build steps required—the page uses vanilla HTML, CSS, and JavaScript.

### Interactions

- Filter posts by tag, keyword search, and sort options (newest, spotlight-first, A–Z)
- Shuffle hero signals to surface fresh tags and a rotating featured drop
- Explore trend radar, spotlight playbooks, and subscribe to the newsletter CTA
